# forum

## usage

flutter安装参考[[flutter](https://flutter.cn/docs/get-started/install)](https://flutter.cn/docs/get-started/test-drive)

- `flutter doctor`检查环境（需要安卓环境）
- `flutter pub get`安装依赖
- `flutter run`运行
